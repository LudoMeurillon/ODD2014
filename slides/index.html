Les instructions usuelles
FROM <image>
Définit l'image de base sur laquelle démarrer le build
RUN
Lance une commande et commit le résultat dans une nouvelle image
exemple : un build maven
ADD <src> <dest>
Copie le fichier <src> dans <dest> au sein du container
exemple : un war
FROM      tomcat
MAINTAINER Ludovic Meurillon <ludovic.meurillon@orange.com>

ADD ./target/monwar.war /tmp

RUN cp /tmp/monwar.war /tomcat/webapps/ROOT.war
